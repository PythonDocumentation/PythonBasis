# 通用转义序列

| 转义序列   | 意义                 | 注释 |
| ---------- | -------------------- | ---- |
| `\newline` | 反斜杠加换行全被忽略 |      |
| `\\`       | 反斜杠(`\`)          |      |
| `\'`       | 单引号(`'`)          |      |
| `\"`       | 双引号(`"`)          |      |
| `\a`       | ASCII响铃(BEL)       |      |
| `\b`       | ASCII退格(BS)        |      |
| `\f`       | ASCII进纸(FF)        |      |
| `\n`       | ASCII换行(LF)        |      |
| `\r`       | ASCII回车(CR)        |      |
| `\t`       | ASCII水平制表(TAB)   |      |
| `\v`       | ASCII垂直制表(VT)    |      |
| `\ooo`     | 八进制ooo码位字符    |      |
| `\xhh`     | 十六进纸hh码位字符   |      |
